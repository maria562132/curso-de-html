Estrutura

O HTML é composto por três componentes básicos que vão dar estrutura para a sua página, são eles: html, head e body.

A tag html é usada para definir qual será o conteúdo da página e como ele deverá ser interpretado, sendo como um contêiner para as outras que se juntam a ele nessa interpretação, possui um começo e um fim.

Dentro da primeira tag entram mais outras duas, onde a head é responsável pelo carregamento de metadados que vão ajudar a entender os outros dados que não são exibidos para o usuário mais facilmente e o body é o corpo do HTML, onde será mostrado todas as informações que estiverem na página para o usuário navegar.

Tags e seus atributos

O que são Tags?

São demarcações dos elementos que estaram presentes dentre de seu documento HTML. Ajudam você com como será organizado e quando for necessário procurar por uma tag são identificadas facilmente.

Possuem um começo e um fim que são representadas pelos símbolos de menor e maior (< >) e de menor, maior e uma barra (< />) ao indicar seu fechamento. Algumas tags não precisam ser abertas e fechadas, são tags chamadas de self-closing que se fecham sozinhas, não necessitando da barra como de costume.

Atributos

São características que vão conrtolar e modificar o comportamento da tag. Existem dois tipos: os globais, que são aceitos por todas as tags e outros são mais específicos para algumas tags.

Exemplo de atributos globais: id, style e class.

Exemplos de atributos específicos: type (usado na tag input), src e href (específicas para links e mídias na tag a e img).

Textos

Algumas tags são usadas especificamente para a tipografia dos textos no HTML, vão os deixar mais organizados e facéis de se entender. São elas: h1 até h6 (são as "headers" ou títulos) e p (parágrafo).

As headers, como no design, são usadas para definir qual texto é mais importante e menos importante, sendo h1 a maior fonte e h6 a menor, são todas em negrito, podendo-se trocar seus valores de cor, fonte e etc.

Os paragráfos já falam por si só, são os textos onde você vai colocar as informações sobre sua página que você deseja que o usuário visualize.

Listas ordenadas e não ordenadas

As listas podem ser usadas de diversas maneiras por você ao fazer sua página de HTML, são compostas por dois tipos: as ordenadas e não ordenadas, possuem uma tag pai e uma tag filha, onde dentro dessa tag pai estará a tag filha.

No caso da lista ordenada, a tag pai será a ol, que irá definir sua lista como uma lista enumerada, ou seja, organizada por números onde serão postos seus conteúdos chamados de itens representados pela tag filha, li.

Já as listas não ordenadas, seguem o mesmo conceito de uma tag pai e uma tag filha mas a única mudandça é que não será definida por números, a ul usa marcadores para definir a lista, que são primeiramente setadas como bolinhas mas podem ser mudadas com o CSS.

Links

Se você for utilizar mais de uma página, imagens ou qualquer tipo de arquivo em seu documento, você necessariamente precisará usar de links para conseguir mostrar o que deseja. Os links são utilizados de diversas maneiras no HTML, mas as mais usadas são para o direcionamento de uma página para outra e na inclusão de imagens.

A tag a será utilizada para que você consiga navegar pelas páginas de seu documento e âncore uma na outra usando um de seus atributos específicos href.

Outra maneira de se utilizar dos links são para incrementar suas páginas com imagens, que podem ser colocadas a partir da tag img e de seu atributo src, onde você pode também mudar seus valores e proporções.
